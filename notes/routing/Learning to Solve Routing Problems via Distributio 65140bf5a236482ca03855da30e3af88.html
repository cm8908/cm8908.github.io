Learning to Solve Routing Problems via Distributionally Robust Optimization

Abstract

  • 현재의 딥러닝 모델들은 한 가지 분포를 가지는 훈련 데이터셋을 사용하기 때문에 일반화의 한계가 있음
  • 본 논문에서는 group DRO (distributionally robust optimization) 방법을 도입해 서로 다른 분포에 대한 모델 파라미터의 joint update 방법을 제시함
  • 여기에 노드 특징을 효과적으로 추출하는 CNN 기반 모듈을 추가로 제시함
  • TSPLib과 CVRPLib에서 일반화 성능 향상을 검증
