Extrude encoder

  • same as topology/geometry encoder but with 71-dim one-hot vector

Extrude decoder

  • same as sketch decoder but only one codebook used and no learnable embedding in cross-attention
  • output is of 72-dim

Generation

  1. Code generation: a Transformer decoder (CodeModel) takes in the obtained code from the encoder and is trained to select code index from the codebook
  1. Sequence generation: Given codes, sketch and extrusion decoder autoregressively generates separate subsequences, and then merging them to form complete output sequence
